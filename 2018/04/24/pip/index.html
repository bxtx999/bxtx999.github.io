  1. pip 指定include 和lib 目录

    在指定目录时需要添加标识 build_ext，例如：

    sudo pip install --global-option=build_ext --global-option="-I/usr/local/include/" --global-option="-L/usr/local/lib" <you package name>

其中-L 指定 lib 文件，
